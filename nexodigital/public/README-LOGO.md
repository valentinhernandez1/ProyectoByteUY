# Logo Byte UY

El proyecto incluye logos en SVG listos para usar:

| Archivo        | Uso                          |
|----------------|------------------------------|
| `logo.svg`     | Header (fondo blanco)        |
| `logo-dark.svg`| Footer (fondo oscuro)        |

Si más adelante querés usar tu propio logo (PNG/SVG), reemplazá estos archivos o actualizá las rutas en `src/config/constants.js` (`LOGO_HEADER` y `LOGO_FOOTER`).
